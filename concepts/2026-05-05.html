In-context learning: how models learn at runtime

Foundations INTERMEDIATE

In-context learning enables large language models to adapt their behavior to new tasks or styles by processing examples provided directly within the input prompt, without altering their underlying weights.

In-context learning is a powerful capability of large language models (LLMs) where they learn to perform a new task or follow a specific format by analyzing examples embedded within the input prompt itself. Unlike traditional machine learning, which involves updating model weights through training or fine-tuning, in-context learning happens entirely at runtime during inference. The model leverages its vast pre-trained knowledge to identify patterns, relationships, and desired output structures from the provided 'few-shot' examples. It then applies this inferred understanding to a new, unseen input within the same prompt. This process does not modify the model's internal parameters; instead, it's about conditioning the model's existing capabilities to respond in a particular way based on the immediate context provided.

Why this matters

Concrete example

Consider using an LLM like `gpt-4` for sentiment analysis. Instead of fine-tuning, you can use in-context learning: `prompt = """Analyze the sentiment of the following reviews: Review: 'This movie was fantastic!' Sentiment: Positive Review: 'The service was terrible.' Sentiment: Negative Review: 'It was okay, nothing special.' Sentiment: Neutral Review: 'Absolutely loved the plot twists!' Sentiment: """` The model, when processing this prompt, identifies the pattern of `Review: [text] Sentiment: [label]` from the first three examples. It then applies this learned pattern to the final review, inferring the task and completing the prompt with `Positive`, without any changes to its internal weights.

Common misconception

Misconception: In-context learning updates the model's internal knowledge base or weights, similar to fine-tuning. Correction: It does not. The model's weights remain static; it merely leverages its existing knowledge to infer patterns from the prompt's examples and apply them to new inputs.
