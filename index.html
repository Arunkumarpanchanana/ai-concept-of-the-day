In-context learning: how models learn at runtime

Foundations INTERMEDIATE

In-context learning allows large language models to adapt their behavior and perform new tasks by processing instructions and examples provided directly within the input prompt, without requiring model weight updates.

In-context learning refers to a large language model's (LLM) ability to learn new tasks or adapt its behavior based solely on the information provided within its input prompt, often including instructions, examples, or specific data. Unlike traditional machine learning, where models learn by updating their internal parameters (weights) through training or fine-tuning, in-context learning happens entirely at inference time. The model leverages its pre-trained knowledge to recognize patterns and relationships presented in the prompt, then applies these patterns to generate a relevant output for a new query. Essentially, the prompt acts as a temporary "working memory" or a set of ad-hoc instructions that guide the model's response for that specific interaction, without making any permanent changes to its underlying architecture or learned capabilities.

Why this matters

Concrete example

Consider using an LLM like `gpt-4` or `Llama-3` for a translation task. Instead of fine-tuning the model on a specific language pair, we can use in-context learning. **Input Prompt:** ``` Translate the following English sentences into French. English: Hello -> French: Bonjour English: Goodbye -> French: Au revoir English: Thank you -> French: Merci English: Please -> French: S'il vous plaît English: How are you? -> French: ``` The model, without any weight updates, learns the translation pattern from the provided examples ("few-shot learning") and completes the last sentence. **Model Output:** `Comment allez-vous ?` This demonstrates how the model "learns" the task and format from the context.

Common misconception

A common misconception is that in-context learning permanently alters the model's weights, similar to fine-tuning. In reality, it does not. The model's internal parameters remain unchanged; the learning is temporary and applies only to the current inference request, leveraging the model's pre-trained ability to follow instructions and recognize patterns.
